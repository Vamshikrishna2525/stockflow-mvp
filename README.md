# StockFlow MVP

A SaaS Inventory Management System built using Spring Boot, React, JWT Authentication, and MySQL.

## Features

- User Signup
- User Login
- JWT Authentication
- Organization-based inventory
- Dashboard
- Product CRUD
- Low Stock Monitoring

## Tech Stack

Backend
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL

Frontend
- React (Vite)
- Axios
- React Router

Deployment
- Backend: Railway
- Frontend: Vercel
